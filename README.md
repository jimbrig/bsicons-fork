An R package for using [Bootstrap icons](https://icons.getbootstrap.com/) in Shiny and R Markdown

## Installation

Install the stable release of `bsicons` on CRAN:

```r
install.packages("bsicons")
```

## Example

``` r
bsicons::bs_icon("geo", size = "3rem")
```
